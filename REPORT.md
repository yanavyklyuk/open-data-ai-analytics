# Звіт про виконання завдання з Git

## Опис проекту
Проект `open-data-ai-analytics` присвячений аналізу відкритих даних "Екологічна та радіаційна обстановка в зоні розташування атомних електростанцій". 

## Виконані кроки:
1. **Ініціалізація:** Створено структуру папок, налаштовано `.gitignore` для ігнорування системних файлів та великих даних.
2. **Feature-гілки:**
   - `feature/data_load`: Реалізовано скрипт завантаження даних.
   - `feature/data_quality_analysis`: Реалізовано скрипт перевірки якості та підготовки даних.
   - `feature/data_research`: Реалізовано скрипт створення моделі та аналізу даних.
3. **Керування конфліктами:** Створено та успішно розв'язано merge-конфлікт у файлі `README.md`, що виник через одночасну зміну вмісту.
4. **Візуалізація та Реліз:** Додано графіки та зафіксовано версію `v0.1.0` за допомогою тегів.

## Висновок
Під час виконання роботи опановано Git Flow, механіку злиття гілок (Merge) та вирішення конфліктів.
