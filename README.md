# Аналіз екологічної та радіаційної обстановки в зоні розташування АЕС України (2018–2025)

### Мета проекту

Дослідити динаміку радіоактивних викидів та скидів українських атомних електростанцій. Основна мета — виявити приховані закономірності в даних моніторингу, перевірити відповідність показників нормам безпеки та побудувати модель машинного навчання для прогнозування або класифікації екологічного стану в зонах розташування АЕС.

### Джерело даних

Дані отримані з Єдиного державного порталу відкритих даних:

* **Набір даних:** [Екологічна та радіаційна обстановка в зоні розташування атомних електростанцій](https://data.gov.ua/dataset/4a9d3d56-bd95-4c3e-97e7-1cdc7bcbd445)
* **Розпорядник:** Міністерство енергетики України

### Питання та гіпотези для аналізу

#### 1. Пошук аномальних відхилень у викидах

* **Питання:** Чи існують у часових рядах точки (квартали), коли рівень радіоактивних викидів був нетипово високим для конкретної АЕС?
* **Гіпотеза:** Показники можуть виходити за межі "нормального" шуму через технічні збої або особливості експлуатації, навіть якщо вони не перевищують гранично допустимі державні норми.

#### 2. Ідентифікація АЕС за «цифровим відбитком»

* **Питання:** Чи можна за хімічним складом викидів (співвідношенням $Cs^{137}$, $Co^{60}$ та йоду) точно визначити станцію без знання її назви?
* **Гіпотеза:** Кожна станція має унікальний профіль викидів, зумовлений типом реакторів (ВВЕР-440 vs ВВЕР-1000), віком обладнання та специфікою систем очистки.

#### 3. Факторний аналіз сумарного індексу викидів

* **Питання:** Які саме типи викидів (інертні гази, цезій чи йод) мають критичний вплив на формування загального індексу викидів (`index_radioactive_releas`)?
* **Гіпотеза:** Загальний екологічний ризик станції визначається не всіма показниками рівномірно, а домінуванням одного-двох специфічних факторів.

### Стек технологій

* **Python** (Pandas, NumPy) — для очищення та підготовки даних.
* **Matplotlib / Seaborn** — для візуалізації трендів та кореляцій.
* **Scikit-learn** — для побудови моделей машинного навчання (регресія, класифікація, детекція аномалій).
